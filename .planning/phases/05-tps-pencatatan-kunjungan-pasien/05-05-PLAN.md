      - Accept `start_date` and `end_date` query params (default to 30 days range: today and 30 days ago if not specified).
      - Query `RekamMedis` to calculate:
        - `total_kasus` (COUNT of visits in the period).
        - `active_kecamatan` (COUNT DISTINCT of kecamatan_domisili in the period).

      - Calculate percentage (`persen`) for each top disease based on `total_kasus`.
      - Construct the JSON payload containing the above stats and a `periode_label` describing the period (e.g., "30 hari terakhir").
      - Return 200 with the payload.
