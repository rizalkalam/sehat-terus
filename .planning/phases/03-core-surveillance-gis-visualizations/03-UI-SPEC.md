## Visual Hierarchy & Focus Points

- **Primary Screen (Surveillance Dashboard)**:
  - **Focal Point**: Interactive Yogyakarta/Jakarta Choropleth Map (colors indicating case density levels).
  - **Secondary Focus**: Region Detail Panel (updating dynamically on sub-district click) and Recharts Time-Series Chart.
  - **Hierarchy Guide**: The map draws primary visual attention through distinct color zoning (Rose/Amber/Emerald). Detail panels and charts provide contextual granular statistics.
- **Accessibility**: All icon buttons (e.g., search, notification, map actions) have text labels or explicit `aria-label` description fallbacks.
